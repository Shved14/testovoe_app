Начальные промпты были сгенерированы ChatGPT, в дальнейшем были доработки основных проблем для корректной работы приложения и удобности использования. Благодаря точным и структурированным промптам, проблемы с SafeArea и базовыми отступами не возникли — изначально было указано использовать SafeAreaView, flexbox и избегать фиксированных высот. Навигация реализована через стек, логика подписки вынесена в отдельный контекст, что позволило корректно управлять состоянием экранов.

Основные сложности возникли при адаптации дизайна под разные размеры устройств. Для средних и больших экранов использовался один вариант компоновки, а для маленьких экранов (например, iPhone SE) был реализован альтернативный layout без вертикального переполнения — с переработанной структурой секции подписок (горизонтальный carousel вместо вертикального списка).

Все изменения тестировались на устройствах с разными размерами экранов для предотвращения overflow и проблем с кликабельностью элементов.
